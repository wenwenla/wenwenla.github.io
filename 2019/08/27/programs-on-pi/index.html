树莓派上的软件们

8月 27, 2019 发布在 软件配置

这里保存若干我的树莓派上的必备软件及其配置。

安装系统

镜像烧录

  1. 格式化软件下载，使用该软件格式化存储卡.
  2. 镜像下载.
  3. 解压镜像，将所有内容复制到存储卡，（注意：你复制进存储卡的应该是若干文件，而不是一个文件夹）
  4. 插上存储卡，开机，图形化安装界面.

更换APT源

更换清华源，按照链接更换为清华源.

中文字体

除去系统需要设置为中文外，需要安装字体. 键入命令：sudo apt-get install ttf-wqy-zenhe

禁止休眠

  1. sudo vim /etc/lightdm/lightdm.conf
  2. 找到#xserver-command=X，当前版本在91行，修改为xserver-command=X -s 0 -dpms

python配置

使用virtualenv，不要升级系统自带的pip3. 键入命令：pip3 install virtualenv -i https://pypi.tuna.tsinghua.edu.cn/simple.
然后，键入命令：python3 -m virtualenv venv.
再然后，键入命令：source venv/bin/active.
之后参照链接更换pip源，在上述命令安装的virtualenv状态下pip和python都是使用的python3.

远程访问

由于家里没有公网IP，因此我通过一台阿里的VPS作为跳板访问家里的树莓派。下文假设VPS的公网IP为1.2.3.4，VPS用户名为vv，树莓派用户名为pi，没有固定的公网IP，但是能够连接互联网。

软件

  • ssh，自带的
  • autossh，用于断线重连

操作方式

先简单尝试下，在树莓派上键入命令：ssh -R 12345:localhost:22 vv@1.2.3.4，然后按照提示输入vv的密码，此时就建立了树莓派到VPS的反向代理，在VPS上对端口12345的访问全部都会转发到对树莓派端口22的访问，由于22是默认的ssh端口，因此这就实现了在任何有网的地方访问家里的树莓派的目的。

试一试在外面访问树莓派，先登录到VPS，也就是在想要访问树莓派的机器上键入ssh vv@1.2.3.4，按提示输入vv的密码。登录上去之后，在vv的终端下键入命令ssh -p12345 pi@localhost，按提示输入pi的密码，即可连接到树莓派上。

尝试之后感觉上述操作过于繁琐，需要先登录VPS然后再登录树莓派，接下来进行一些操作，实现一步登录到树莓派的目的。

通过ssh登录到VPS上，修改文件/etc/ssh/sshd_config，在最后加上GatewayPorts yes，然后通过指令sudo service sshd restart重启sshd服务。登录到树莓派上，输入命令ssh -R 12345:localhost:22 vv@1.2.3.4。之后便能通过命令ssh -p12345 pi@1.2.3.4直接登录树莓派（注意这边提示输入密码时输入的是树莓派的密码）。这一步执行完毕后如果还是无法直接连接到树莓派，登录到VPS上，执行命令lsof -i12345，查看端口12345的占用，应当能够看到类似TCP *:12345 (LISTEN)的输出，如果不是*而是显示localhost则是操作没有成功，重启一下试试。否则，检查VPS的防火墙，包括ufw命令以及VPS网页版控制台的防火墙，或者重启一下？

搞完之后觉得美滋滋，但是ssh在一段时间没有操作之后会自动断掉，一旦断掉了就只能回家的时候再连接上了，这可真惨。因此我使用autossh工具实现断线自动重连。

首先，使用命令sudo apt install autossh安装autossh，使用命令autossh -M 12346 -NR 12345:localhost:22 vv@1.2.3.4来实现不断线的ssh，这里的12346是用于监控的端口，只要保证能通过防火墙，并且没被其他程序占用即可。每次输入命令很麻烦，把它加入开机自启服务，在此之前我们得先实现免密登录。使用ssh-copy-id直接把公钥传过去就行了。

其他操作

  • 厌倦了命令行，想使用图形界面咋办？ssh -R 12345:localhost:5900 vv@1.2.3.4转发VNC端口流量就行啦注意VNC连接设置中，端口写12345，IP写1.2.3.4
  • 同理，也可以转发80等端口的流量，这个随便操作一下就行。
  • 在外面突然连不上了，首先考虑家里断网了。其次，autossh每隔10分钟会尝试重连，稍微等等。如果长时间连不上，可能是VPS上端口一直被占用，登上VPS，lsof -i:12345查一下端口被哪个进程占用着，kill掉，然后应该过个几分钟就好了（家里没断网的话）。
